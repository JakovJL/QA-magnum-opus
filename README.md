# Knowledge Base

This is an Obsidian vault for studying Manual QA and AQA with Java.

The vault is built around the Innowise trainee programme. It contains theory notes, interview practice, examples, flashcards, and reference materials.

English notes are the main version. Russian notes are mirror translations.

## Use Obsidian

This knowledge base should be opened and used in Obsidian.

Obsidian is needed because the vault uses:

- internal links like `[[QA manual eng]]`
- links to sections like `[[#Table of Contents]]`
- the graph view
- flashcards
- Markdown notes with Obsidian-specific formatting

To use the vault:

1. Open Obsidian.
2. Choose **Open folder as vault**.
3. Select the `knowledge base` folder.
4. Work with notes inside Obsidian, not only through a file explorer.
